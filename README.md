# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer, get instant results, and keep track of your score in a clean and responsive interface.

## 🚀 Features

- 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Random computer choice
- 📈 Live score updates
- 🎯 Instant win, lose, or draw result
- 💻 Simple and user-friendly interface
- 📱 Responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript 

## 📂 Project Structure

```
rock-paper-scissors-game/
│── ht.html

│── paper.png
│── scissors.png
```

## 🎮 How to Play

- Click on **Rock**, **Paper**, or **Scissors**.
- The computer randomly selects its move.
- The winner is decided based on the game rules:
  - Rock beats Scissors
  - Scissors beats Paper
  - Paper beats Rock
- The result and score are updated after every round.
